CSS Selectors Cheat Sheet

Basic Selectors

Selector Example Description
Class Selector .class-name Selects all elements with the specified class.
ID Selector #id-name Selects the element with the specified ID.
Element Selector element Selects all elements of the specified type.
Attribute Selector [attribute=value] Selects elements with the specified attribute and value.

Combinator Selectors

Selector Example Description
Descendant Selector ancestor descendant Selects all descendants of the specified ancestor.
Child Selector parent > child Selects all direct children of the specified parent.
Adjacent Sibling Selector previous + next Selects the next sibling immediately following the specified element.
General Sibling Selector element ~ siblings Selects all siblings following the specified element.

Pseudo-class Selectors

Selector Example Description
:hover element:hover Selects elements when the user hovers over them.
:first-child element:first-child Selects the first child of the specified element.
:nth-child(n) element:nth-child(n) Selects the nth child of the specified element.
:not(selector) :not(selector) Selects elements that do not match the specified selector.
